✅ STEP 2: RUN THE SETUP (Choose ONE)

OPTION A: Windows Users (Easiest!)
───────────────────────────────────
1. Double-click: SETUP-ALL.bat
   (Installs dependencies)
   
2. Wait for completion (see "Press Enter to exit")

3. Double-click: START-ALL.bat
   (Starts all services)

4. Done! Three windows will open:
   • Backend server (terminal 1)
   • Frontend app (terminal 2)
   • Information display

OPTION B: Command Line (Any OS)
────────────────────────────────
PowerShell or Terminal:
   ./SETUP-ALL.ps1
   ./START-ALL.ps1

Manual (If scripts don't work):
   cd server && npm install
   cd ../web && npm install
   # Then start in separate terminals:
   # Terminal 1: cd server && npm run dev
   # Terminal 2: cd web && npm run dev

✅ STEP 3: LOAD THE CHROME EXTENSION

After services are running:

1. Open Google Chrome
2. Go to: chrome://extensions/
3. Find the toggle in top-right corner
   → Turn ON "Developer mode"
4. Click "Load unpacked" button
5. Navigate to and select: extension/ folder
6. Done! Extension is now loaded

Test it:
   • Visit Amazon.com, Walmart.com, Daraz.pk, or BestBuy.com
   • Click the extension icon
   • Try "Compare price" or "Apply coupon"
